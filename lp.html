Problem

ドライブ後の3つの悩み

楽しかった旅も、帰り道のお金の計算で気まずくなっていませんか？

😓
傾斜計算が面倒
運転手に多く負担してほしいけど、計算が難しくて結局均等割りに。気を使って損してしまう。
🛣️
ETC料金がわからない
高速料金はあとで明細を見るまで正確にはわからない。その場でみんなに説明できない。
💸
集金が遅れる
「あとで払う」が続いて、結局運転手がずっと立て替えたまま。催促するのも気まずい。
Features

Warikan Drive が全部解決

出発前に全員の負担金額を確定。その場で集金まで完結。

FEATURE 01
🗺️
経路 × 料金を自動取得
Google Maps Routes APIで複数経路を比較。ETC/現金・時間帯割引（深夜・休日）を出発時刻から自動反映。
FEATURE 02
🚗
運転手傾斜割り勘
スライダーで運転手の負担係数を設定。ガソリン代 + 高速代 + 追加費用をまとめて公平に計算。
FEATURE 03
📅
日別 × 複数ルート管理
旅行ごとに日付・ルートを管理。駐車場・レンタカー・カーシェア代も合算。複数日の旅行も対応。
FEATURE 04
📱
Google Maps ナビ連携
選択したルートをそのままGoogle Mapsで開いてナビ起動。アプリを切り替える手間ゼロ。
FEATURE 05
💴
PayPay 即時請求
計算結果から即座にPayPay請求リンクを発行。メンバー全員に一斉送信で集金を完結。
FEATURE 06
🌙
ダーク / ライトモード
夜のドライブでも目に優しいダークモード搭載。スマホ最適化で車内でもサクサク使える。
How it works

使い方は 4 ステップ

難しい設定は不要。目的地を入れるだけで、あとは全自動。

📍
ルート入力
出発地・目的地・出発時刻を入力
🔍
経路選択
複数候補から最適なルートを選ぶ
⚙️
人数・設定
人数・燃費・運転手優遇率を設定
💰
割り勘 完了
全員の負担金額が一発で確定
計算例：東京駅 → 箱根（4人・ETC・運転手優遇70%）
🗺 東京駅 → 箱根 · 95.2 km
